PART 2: PDF export fix

The PDF export is failing because Helvetica does not support some Unicode characters.

Fix the PDF export by sanitizing text before writing to PDF.

Create a helper function:

def sanitize_pdf_text(text):
    if text is None:
        return ""
    text = str(text)
    replacements = {
        "•": "-",
        "–": "-",
        "—": "-",
        "“": '"',
        "”": '"',
        "‘": "'",
        "’": "'",
        "…": "...",
        "→": "->",
        "≤": "<=",
        "≥": ">=",
    }
    for old, new in replacements.items():
        text = text.replace(old, new)
    return text

Apply sanitize_pdf_text() to every text string before writing it into the PDF.

This includes:
- Titles
- Section headers
- KPI labels
- Recommendation text
- Decision labels
- Methodology text
- Pattern analysis text
- Any bullet list text

Also replace bullet lists in the PDF with simple hyphen lines instead of bullet characters.

Example:
Instead of:
• Scale C05
Use:
- Scale C05
